About Feedback App

Our Mission

We provide a secure, intelligent feedback platform that helps businesses and organizations understand their users better through advanced sentiment analysis and comprehensive feedback management.

Key Features

  • Secure Authentication: JWT-based authentication with secure cookies
  • AI-Powered Sentiment Analysis: VADER and Hugging Face Transformers integration
  • Real-time Analytics: Instant sentiment preview and detailed statistics
  • Role-Based Access: User and admin roles with appropriate permissions
  • Data Export: CSV export functionality for analysis

Technology Stack

Backend
  • Flask 3.0.0
  • SQLAlchemy ORM
  • PostgreSQL / SQLite
  • JWT Authentication
  • Rate Limiting
Frontend

  • Bootstrap Icons
  • Vanilla JavaScript

  • Dark/Light Mode
